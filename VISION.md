# vision

a hub for continuously creating in public. the site is a tree: a seed growing into
a grand one — trunks, branches, leaves, fruit. it only grows. everything on it is
simple to the point of nearly bare, and every remaining choice exact. functional,
graceful, never overly complex.

## the trunks (fixed, five)

- **work** — finished, useful things. experiments graduate here.
- **writing** — words. home of the practice.
- **experiments** — ideas in progress; living in public.
- **sound** — mixes and masters, posted as they evolve.
- **curation** — art, objects, websites, people, music, text worth keeping.

## the branches

- **the practice** (writing) — one post every day, seth godin style.
- **the log** (experiments) — habit tracking (reading, meditation, exercise) and
  daily working hours, rendered beautifully.
- **predictions** (experiments) — public predictions with confidence; over time,
  a calibration score shows real forecasting ability.
- **trading sim** (experiments) — practice trading on pure synthetic price action;
  drill sub-skills: swings, scalps, compounding, taking profit.
- **mixes & masters** (sound) — every version of every mix, progress in public.
  visitor feedback/scoring: deferred until earned (tiny worker later).
- **more data ideas** — parked. only build when a real one appears.

## constraints (non-negotiable)

- no build step, no framework, no dependencies. plain html/css/js.
- the agent is the static site generator: content files and their json indexes
  are written by hand each session, per the recipes in CLAUDE.md.
- git is the database: all logs and content live as files in this repo.
- one accent (`--accent`), near-monochrome everything else. lowercase copy.
- mono (`--mono`) for marks, labels, and data. one load reveal max.
- accessible floor: mobile responsive, `:focus-visible`, reduced-motion, contrast.

## graduation rule

an experiment that proves genuinely useful moves under /work/, leaving a stub
redirect behind. the tree never loses a branch; it thickens.

this file changes rarely. ROADMAP.md changes every session.
